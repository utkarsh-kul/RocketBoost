# RocketBoost

RocketBoost is a 3D arcade-style rocket navigation game developed in Unity. Players control a rocket using realistic thrust mechanics while avoiding moving obstacles and navigating challenging levels. It is a Unity project I created while learning game development. I expanded on the core mechanics by refining the gameplay, improving the visuals, and polishing the overall experience.

## Gameplay

Fly the rocket from the launch pad to the landing platform without colliding with obstacles.

## Features

- Physics-based rocket movement
- Collision detection
- Moving and rotating obstacles
- Particle effects
- Sound effects
- Multiple handcrafted levels
- Stylized sci-fi environment

## Built With

- Unity 6
- C#
- Visual Studio

## Controls

- Space - Thrust
- A/D - Rotate
